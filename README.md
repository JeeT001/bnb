Ongaonga Bed & Breakfast – Web Application

A full-stack PHP web application developed as part of a Web Programming project, based on a real-world Bed & Breakfast business brief. The application supports customer management, room management, and booking workflows.

⸻

Features
	•	Customer registration and login
	•	View available rooms
	•	Create, update, and delete bookings
	•	Date-based room availability checking
	•	Admin-style management of customers, rooms, and bookings

⸻

Tech Stack
	•	PHP
	•	HTML, CSS
	•	MySQL (phpMyAdmin)
	•	Git and GitHub

⸻

Key Concepts
	•	CRUD operations using PHP and MySQL
	•	Database relationships and constraints
	•	Form handling and validation
	•	Manual testing with test plans and bug tracking

⸻

Setup (Local)
	1.	Clone the repository
	2.	Run the project using a local PHP server (e.g. XAMPP)
	3.	Import the database using phpMyAdmin
	4.	Update database connection settings if required

⸻

Project Context

Developed for learning and demonstration purposes, based on a provided business design brief.
